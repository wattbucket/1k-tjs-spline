# flask-parcel-starter
Starter for Flask apps using parcel-bundler for bundling.

## desarrollo
### terminal js
```js
npm i
npm run watch
```

### terminal py

```py
python3 -m venv ./.venv;
source ./.venv/bin/activate;
pip install -r requirements.txt

```

```py
python app.py
```
